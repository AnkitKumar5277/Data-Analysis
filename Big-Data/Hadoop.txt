HDFS 
(Hadoop Distributed File System) ek distributed storage system hai jo bade data ko multiple machines par store karke efficiently manage karta hai.

MapReduce 
MapReduce data ko process aur analyze karne ke liye use hota hai.
MapReduce ek programming model hota hai jo large data ko parallel processing ke through "Map" (filtering & sorting) aur "Reduce" (summarizing) steps mein process karta hai.
